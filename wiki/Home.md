PromptFx is designed to help you (i) rapidly create and test prompts and other generative AI tools, and (ii) use generative AI for basic local applications such as PDF document Q&A, translation, and more.

## Installing and Running PromptFx

1. **Download a release** from the [📦 Releases page](https://github.com/aplpolaris/promptfx/releases).
2. **Extract the ZIP** for your platform:
    - `promptfx-<version>-windows.zip`
    - `promptfx-<version>-macos.zip`
    - `promptfx-<version>-linux.zip`

3. **Add an API key**. See `README.md` in the unzipped folder for specific instructions.

4. **Run the launcher script**:
    - **Windows**: Double-click `run-windows.bat`
    - **macOS**: Right-click `run-macos.command` → Open (you may need to approve it in Security & Privacy)
    - **Linux**: Run `./run-linux.sh` in the terminal (you may need to `chmod +x` first)

> You’ll need Java 17+ installed unless you’re using a bundled version (coming soon).

See `README.md` in the unzipped folder for additional instructions, and [UI Overview](https://github.com/aplpolaris/promptfx/wiki/UI-Overview) for a guide to features and functionality.
