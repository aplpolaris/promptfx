# MCP Servers View
Displays MCP server capabilities, as configured in `config/mcp-servers.yaml`. Currently this is view-only.

<img src="https://github.com/user-attachments/assets/a41f43b1-32c6-4eee-9051-c003a97f0e42"/>

# MCP Prompts View
The prompts view shows the collection of prompts across all configured MCP servers. It displays the list of fields, bolding required fields, and generates the filled prompt on request.

<img src="https://github.com/user-attachments/assets/8a6c8b55-3e46-49e8-827c-96aa9097d74c" />

# MCP Resources View
The resources view shows the collection of resources across all configured MCP servers. You can use `resources/read` to view resources provided by MCP servers.

<img src="https://github.com/user-attachments/assets/6337620e-ccda-4e58-ab44-81fdd112e621" />

# MCP Tools View
The tools view shows the collection of tools across all configured MCP servers. You can use `tools/call` with custom JSON parameters to try out the tool.
